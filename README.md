## Usage


Preinstalled tools:
- CDK
- AWS CLI (optional)


After cloning:

```
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements-dev.txt
```

Deploy:

```
$ cdk deploy
```
